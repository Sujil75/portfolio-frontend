# рҹҡҖ Personal Portfolio

A modern, responsive, and dynamic portfolio built with **React + Vite**. The application showcases my professional profile, technical skills, education, projects, and contact information by consuming data from a dedicated backend API. The project follows a component-based architecture with reusable hooks, global state management, and responsive UI.

## вҡҷпёҸ Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd portfolio-frontend
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build the project

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
